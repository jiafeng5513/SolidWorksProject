My Voron Mods
=============

## 文件说明
1. `docs`: 文档和打印机配置文件备份
2. `PSU-bracket`: 电气导轨卡扣
3. `Stealthburner+Orbitu2.0+ebb36`: 热端mod相关
4. `电器仓固定件`: 带断路器的裙边和一些电器仓固定散件
5. `料架`: 耗材卷轴
6. `排风管`: 打印仓后方毒气排放风管
7. `前门折页`: 前门大角度折页
8. `Voron3D打印机软件维护指南.md`: Voron2.4+EBB Can Bus软件维护备忘录
